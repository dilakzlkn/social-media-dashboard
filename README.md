# Social Media Addiction Dashboard

This project is an interactive dashboard built with **R Flexdashboard** using the **Students Social Media Addiction** dataset.

It explores students’ social media usage habits and addiction levels through summary tables and visualizations.

---

## Live Dashboard

[Click here to open the dashboard](https://dilakzkn.github.io/social-media-dashboard/)

## Project Files

- `Students-Social-Media-AddictionDashboard.Rmd` → Dashboard source code (R Markdown)
- `Students Social Media Addiction.csv` → Dataset used in the analysis
- `index.html` → Main dashboard page (GitHub Pages)
- `Students-Social-Media-AddictionDashboard.html` → Generated dashboard output
